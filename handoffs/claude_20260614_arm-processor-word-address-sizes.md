## Current state
- Question 2 (CMN) answered from the TRM:
  - Max Physical Address (PA) width = **52 bits** (`Arm_Neoverse_CMN_S3AE_TRM_r2p5.md:307`).
  - `PA_WIDTH` configurable: 34, 44, 48, 52 (default 48); `REQ_ADDR_WIDTH` (CHI REQ-flit ADDR field): 44, 48, 52 (default 48), must be >= `PA_WIDTH` (`Arm_Neoverse_CMN_S3AE_TRM_r2p5.md:681-682`).
