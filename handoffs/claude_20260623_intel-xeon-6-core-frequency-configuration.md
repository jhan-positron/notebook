## Current state
- Question answered and doc-backed. Scenario (A) = SST-TF (Turbo Frequency, marketed as Priority Core Turbo) layered on SST-CP CLOS: bucket-0 on this SKU = 8 high-priority cores per package -> 4400 MHz, low-priority cores clipped to 2700 MHz SSE (AVX2 2400 / AVX-512 2100 / AMX 1900).
