  from CPUID by default; tsx=on re-enables via IA32_TSX_CTRL. Absence of
  rtm in /proc/cpuinfo + TAA "Not affected" does not mean the silicon
  lacks it.
- The Claude Windows desktop app's Code tab has no /mcp command — MCP
  OAuth cannot be completed in-app; the CLI is required.

  the Windows-box .jsonl files.
