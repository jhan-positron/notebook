  with refinement: Arm CMN-S3 CHI mesh does snoop-filter-directed
  snooping at HN-F home nodes (not broadcast). C1-Nano (Nevis) cores in
  DSU clusters attach as RN-F (2 per DSU); HN-F nodes host SLC + snoop
  function per SAM hemisphere slice; DMA masters (4 SPM/OCM DMA engines
  north edge, 2 DMA-350s south edge, host PCIe) enter via RN-I ports over
  ACE5-Lite->AXI4 bridges. Caveats: one-way I/O coherency (nothing snoops
  the cache-less DMA engines); snooping is per-transaction
  attribute-dependent (AxDOMAIN/AxCACHE -> CHI ReadOnce/WriteUnique vs
  ReadNoSnp/WriteNoSnp; the SPM leg behind SBSX is deliberately
  non-coherent); SLC-as-OCM acts as a DDIO-like DMA landing zone.
  Conclusion: no dma_sync_*-style maintenance needed for coherent
  address spaces.
