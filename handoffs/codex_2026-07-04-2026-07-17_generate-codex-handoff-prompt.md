## Gotchas & decisions

- The Codex app may expose remote cwd groups via thread inventory that are not
  visible as saved Projects. Treat them as scope candidates and group by
  host/cwd if no Project label exists.
- `session_index.jsonl` records chat title changes over time. Use newest title
  for `Codex chat:` and preserve old names in `Formerly named:` lines.
- `SCOPE: auto` is intentionally Codex-first. Scanning GitHub first misses brand
  new chats and was removed from the default behavior.
- Repo artifact copies are mirrors. Edit canonical workspace files first when
  the canonical still exists, then refresh the repo copy.
- GitHub blob pages do not render standalone HTML. Use the
  `htmlpreview.github.io/?https://raw.githubusercontent.com/...` rendered URL in
  the HTML source, visible page link, and artifact README.
